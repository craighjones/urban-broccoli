A design system for Google's Web Fundamentals

Overview

An aggressive delievery to design and produce a scalable design system in code within 6 weeks. The project team adopted a lean delivery methodology, which embedded design and tech into the 3 sprints embracing pair design and collaboration.

Design was light, producing a minimum set of components and patterns, so they could be built and iterated in the browser.
